Command Line

  RED+.exe [SWITCHES] "path"
      
    -autosearch is an OPTIONAL switch which searches the supplied path as soon 
    as RED+ starts. If supplied it MUST be before "path"
      
   "path" is the directory to be searched. It should be enclosed in double quotes.

e.g.RED+.exe -autosearch "X:\NotBob\Test"
